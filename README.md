# 🏡 House Price Prediction Database Analysis

## 📌 Project Overview

The **House Price Prediction Database Analysis** project demonstrates end-to-end database design and SQL analysis for a real estate business scenario. The project includes database modeling, data generation, relational database design, SQL analysis, and business reporting.

The dataset contains realistic property, builder, owner, agent, sales, and location information, enabling comprehensive business analysis using SQL.

---

## 🎯 Project Objectives

* Design a normalized relational database.
* Generate realistic housing datasets using Python.
* Import CSV datasets into MySQL.
* Create relationships using Primary Keys and Foreign Keys.
* Validate imported data.
* Perform business analysis using SQL.
* Create reusable SQL Views for reporting.
* Build a Power BI dashboard for business insights.
